DevOps to połączenie Rozwoju (Dev) i Utrzymania (Ops).

### Filozofia CAMS

1. **Culture:** Wspólna odpowiedzialność (Blame-free culture).
    
2. **Automation:** CI/CD (Continuous Integration / Deployment).
    
3. **Measurement:** Monitorowanie wszystkiego (SLI, SLO, SLA).
    
4. **Sharing:** Współdzielenie wiedzy i narzędzi.

    Plan --> Code --> Build --> Test --> Release --> Deploy --> Operate --> Monitor
    Monitor --> Plan

- **Testowanie:** Selenium, JMeter.
    
- **Operacje:** JIRA, Ansible.
    
- **Monitoring:** ELK Stack, Splunk.
    
- **Inżynieria Chaosu:** Narzędzia typu **Netflix Simian Army** (Chaos Monkey) – celowe wyłączanie serwerów na produkcji w celu testowania odporności.
